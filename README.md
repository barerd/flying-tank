This is a project to help me learn C, since Common Lisp (my goto language to protottype) is not working well with SDL, and GODOT and UNITY are way too overweight for me at the moment. Currently, the game has a Tank entity, which can be moved with up arrow key, and turned with left and right arrow keys. Also a red rectangle is present (controlled with WASD) to allow collision detect. I am currently battling with making the exhaust flame appear correct, as I want to learn how to decorate entitties with other entities, which may be still images or animations. Example are turrets, missile launchers, etc. The main idea (and struggle) is to get a working framework to handle attachment transforms and control flow between parent and child entities. So that a doctor image can hold a stethoscope, etc.
